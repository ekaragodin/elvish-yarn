# elvish yarn

Completions for `yarn`.

## Usage

Install the `elvish-yarn` package using epm:

```
use epm
epm:install github.com/ekaragodin/elvish-yarn
```

In your `rc.elv`, load this module:

```
use github.com/ekaragodin/elvish-yarn/yarn
```
